# 즐거운 개발을 위한 설정사항
(1) 커밋 메시지 템플릿 설정하기.
repo root에 .gitmessage.txt 커밋 메시지 템플릿 파일이 있습니다.
이를 활용하기 위해서는 다음의 명령어를 repo root에서 실행하셔야 합니다.
```
git config commit.template ./.gitmessage.txt
```
